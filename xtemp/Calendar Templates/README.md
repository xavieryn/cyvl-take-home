# Calendar Templates

Realistic Google Calendar events for the AI Chief of Staff demo.
Each file represents one calendar event the CEO would have.

## How to use
Add these to Google Calendar manually or ask Claude to send them via the Calendar API.

## Categories
- CONFLICT — two events overlapping (system should flag these)
- BACK-TO-BACK — no buffer between meetings (system should flag)
- PREP-NEEDED — complex meeting with no prep block (system should suggest one)
- TRAVEL — travel days affecting availability
- RECURRING — regular cadence meetings
- FOCUS — deep work blocks
- EXTERNAL — customer/investor/press calls
- INTERNAL — team meetings
- PERSONAL — personal appointments mixed into work calendar
