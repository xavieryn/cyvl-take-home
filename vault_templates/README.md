# CYVL Chief of Staff — Vault Guide

This vault is managed by the AI Chief of Staff system. Most notes here are auto-generated from your Gmail and Google Calendar each morning.

---

## Folder Structure

```
CYVL Chief of Staff/
├── Daily Briefings/     ← Start here every morning
├── Meeting Notes/       ← Auto-created before each meeting
├── People/
│   ├── Investors/       ← One note per fund/partner
│   ├── Clients/         ← One note per municipality or enterprise
│   └── Team/            ← Internal team members
├── Deals/               ← Active sales opportunities
├── Projects/            ← Ongoing initiatives
├── Tasks/               ← High-priority action items (🔴 urgent only)
├── Company/
│   ├── OKRs/            ← Quarterly objectives + key results
│   ├── Decision Log/    ← Major decisions + rationale
│   └── Retrospectives/  ← Post-mortems
└── Templates/           ← Source templates (don't edit these directly)
```

---

## Daily Workflow

1. Open **Daily Briefings/** → today's date → read the executive summary
2. Check **Tasks/** for any new high-priority action items
3. Use the leadership task board for CEO-level decisions

---

## Templates

All templates are in `Templates/`. Use Obsidian's template inserter (`Ctrl/Cmd+P → Insert Template`) to create a new note from a template.

| Template | Use for |
|----------|---------|
| Meeting Notes | Any meeting — agenda, decisions, action items |
| Task Board | Team-level weekly task tracking |
| City-Client | Each municipality CYVL is selling to |
| Deal | Each active sales opportunity |
| Person | Investors, key contacts, partners |
| Weekly Review | Friday end-of-week wrap |
| Project | Any multi-week initiative |
| OKR | Quarterly goal-setting |
| Decision Log | Major decisions you want to be able to revisit |
| Investor Note | Per-fund fundraising tracker |
| Hire Tracker | Per-role hiring pipeline |
| Retrospective | Post-mortems on deals, launches, incidents |
| Daily Note | Personal daily planning alongside the AI briefing |
